```

Requirements:
- Target audience: [Age group, interests]
- Tone: [Engaging/Motivational/Mysterious/Educational]

- Structure: Clear introduction, 3-5 main points, strong conclusion
- Include pattern interrupts every 60 seconds
- Add music cues: [MUSIC: Suspenseful], [MUSIC: Upbeat], etc.
- Add visual cues: [VISUAL: Show image of...], [TEXT ON SCREEN: ...]
- Keep sentences short and punchy
- End with strong CTA (like, comment, subscribe)
- Word count: 1,200-1,500 words (for 8-10 min video)

Script should be ready for voiceover - no stage directions in speech."
```

### STEP 5A: Create Video with FREE TOOLS (CapCut Method)

**What You Need:**
- CapCut (Free - download on phone or PC)
- Your voiceover MP3
- Stock footage from Pexels/Pixabay

**Production Process:**

**1. Gather Visual Assets (15 minutes)**

Go to:
- **Pexels.com** - Free stock videos
- **Pixabay.com** - Free videos and images
- **Coverr.co** - Free B-roll footage

Search terms based on your script:
- If script mentions "money" → search "money, cash, wealth"
- If script mentions "success" → search "business, entrepreneur, office"
- Download 20-30 clips (more than you need)

**2. CapCut Desktop/Mobile Editing:**

**Import:**
- Open CapCut
- Import your voiceover
- Import all video clips

**Timeline:**
- Drag voiceover to timeline
- Listen to first sentence
- Add matching video clip above voiceover
- Cut clip when sentence ends
- Repeat for entire video

**Quick Cuts:**
- Change video every 3-5 seconds
- Keeps viewers engaged
- Use "Split" tool constantly

**Add Text:**
- Use "Text" → "Default"
- Add key phrases on screen
- Yellow text with black outline (most readable)
- Position: Lower third

**Transitions:**
- Select clip → "Transition"
- Use: Dissolve, Slide, Zoom
- Duration: 0.3-0.5 seconds

**Background Music:**
- Go to "Audio" → "Music"
- Choose no-copyright music
- Lower volume: 15-20%
- Don't overpower voiceover

**Export Settings:**
- Resolution: 1080p (minimum)
- Frame Rate: 30fps
- Format: MP4

### STEP 6: Create Killer Thumbnail (10 minutes)

**Go to Canva.com (Free)**

**Size:** 1280 x 720 pixels (YouTube standard)

**Thumbnail Formula:**
```
Bold Text + Emotion Face/Object + Bright Colors + Curiosity Gap
```

**ChatGPT Prompt for Thumbnail Ideas:**

```
"Generate 5 YouTube thumbnail concepts for my video titled: [YOUR TITLE]

For each concept, describe:
- Main visual element
- Text overlay (3-5 words max)
- Color scheme
- Emotional trigger
- What makes it clickable

Style: High contrast, attention-grabbing, mobile-friendly"
```

**Canva Tutorial:**
1. Open Canva → "YouTube Thumbnail"
2. Use template OR create custom
3. Add background image from Pexels
4. Add text:
   - Font: Anton, Bebas Neue, or Impact
   - Size: LARGE (readable on phone)
   - Colors: Yellow, Red, or White
   - Stroke: Black outline (always!)
5. Add arrow/circle to highlight key element
6. Export as PNG

**Pro Thumbnail Tips:**
- Show emotion (shocked face, money, success symbols)
- Use contrast (dark background + bright text)
- Keep it simple (3-5 words max)
- Test on your phone (does it look clear when small?)

## PHASE 5: EXACT PROMPTS FOR DIFFERENT NICHES

### Motivational/Success Content

**Script Prompt:**
```
"Write a 10-minute YouTube script: 'How [Successful Person] Built [Achievement] 
From Nothing'

Structure:
- Hook: Their biggest failure/rock bottom moment (30 seconds)
- Early life: Humble beginnings (2 min)
- Turning point: The decision that changed everything (2 min)
- Strategy 1: First key to success (1.5 min)
- Strategy 2: Second key to success (1.5 min)
- Strategy 3: Third key to success (1.5 min)
- Current status: Where they are now (1 min)
- Lesson: What you can learn (30 seconds)
- CTA: Inspire action (30 seconds)

Tone: Inspirational but authentic
Include: Specific numbers, dates, quotes
Add: [MUSIC] cues for emotional moments"
```

### True Crime/Mystery

**Script Prompt:**
```
"Write a chilling 12-minute YouTube script about 'The [Crime/Mystery Name]'

Structure:
- Cold open: Most shocking moment (30 seconds)
- Setting: Time and place (1 min)
- Victim/Subject: Who they were (1.5 min)
- Timeline: Day of incident (3 min)
- Investigation: What police found (2 min)
- Suspects/Theories: Main theories (2 min)
- Plot twist: Unexpected detail (1 min)
- Current status: Solved or unsolved? (1 min)
- Reflection: Unsettling conclusion (30 seconds)

Tone: Dark, suspenseful, respectful
Include: Verified facts only
Add: [MUSIC: Tense] and [PAUSE: 2 seconds] cues"
```

### Educational/Explainer

**Script Prompt:**
```
"Create an 8-minute explainer script: 'How [Complex Topic] Actually Works'

Structure:
- Hook: Why this matters to viewer (30 seconds)
- Common misconception: What people get wrong (1 min)
- Simple explanation: Core concept (2 min)
- Real example 1: Practical application (1.5 min)
- Real example 2: Another angle (1.5 min)
- Common mistakes: What to avoid (1 min)
- Action steps: How to use this knowledge (30 seconds)
- CTA: Encourage questions in comments (30 seconds)

Tone: Friendly teacher
Use: Analogies, avoid jargon
Target: Complete beginners"
```
